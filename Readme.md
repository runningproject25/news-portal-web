## 📌 About This Project

A full-stack **web information portal** for Politeknik Negeri Ujung Pandang (PNUP),
built from scratch with a clean and responsive design.

### ✨ Features
- 🔐 Secure admin authentication with session & token-based access
- 📰 Full CRUD — manage news, announcements, study programs & registration forms
- 🖼️ Image upload support for articles and announcements
- 📋 Dynamic study program table with accreditation info
- 📱 Responsive design using Bootstrap 5
- 🗄️ MySQL database with structured relational data
